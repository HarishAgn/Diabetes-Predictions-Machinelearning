# Diabetes Prediction Using Machine Learning

## Project Overview
This project predicts whether a patient is diabetic based on medical attributes using Machine Learning.

## Objectives
- Predict diabetes accurately
- Analyze important health features
- Compare actual and predicted results

## Dataset
The dataset contains medical information such as:
- Pregnancies
- Glucose
- Blood Pressure
- Skin Thickness
- Insulin
- BMI
- Diabetes Pedigree Function
- Age
- Outcome
